
## webpack + angular + oclazyload + express (devmiddleware + hotmiddleware)
* angularjs1.x + oclazyload 按需加載
* webpack1.x  代码合并压缩
* express  node server

## learning guide
* angular1
* webpack
* express
* oclazyload

## 环境要求
* node
* python (编译sass)

## 安装/开发/发布
* 安装命令 (npm install)
* 编译文件作为web独立文件 (npm run prod)
* 发布 (npm start)
* 开发模式 (npm run dev)
